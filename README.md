#  Android 淘宝 爬虫 学习
### 介绍
> 用于征信、金融类型APP的淘宝数据获取(昵称、手机号码、正式姓名、性别、生日、身份证、登录邮箱、极速退款金额、头像地址、成长值、VIP等级、淘气值、
是否实名认证、淘宝地址、订单详情、支付宝账号、账户余额、余额宝余额、累计收益)
